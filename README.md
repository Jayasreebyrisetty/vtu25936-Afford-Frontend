# Campus Notification System (Frontend)

## 📌 Project Overview
This is a React-based Campus Notification System built for frontend evaluation.  
It displays notifications fetched from an API and organizes them using a priority-based system.

---

## 🚀 Features
- User Authentication (Login & Register using localStorage)
- Protected Dashboard route
- Fetch notifications from API
- Priority Inbox system
  - Placement (High Priority)
  - Result (Medium Priority)
  - Event (Low Priority)
- Sort by priority and latest timestamp
- Display top 10 notifications
- Responsive UI using Material UI

---

## 🛠️ Tech Stack
- React (Vite)
- JavaScript (ES6)
- React Router DOM
- Material UI
- Fetch API
